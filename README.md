QuickChatA multi-threaded LAN chat application built in Java, covering core concepts from a Java programming course — sockets, threads, Swing GUI, and file I/O.What's builtArchitecturesrc/
├── server/
│   ├── ChatServer.java       — TCP server, channel routing, admin console
│   ├── ClientHandler.java    — tracks user channel state on its own thread
│   └── ChatLogger.java       — thread-safe, daily rotating log writer
└── client/
    ├── ChatClient.java       — CLI client with terminal auto-reconnect loop
    └── ChatClientGUI.java    — Swing desktop client with file transfer UI
FeaturesDrag & connect — Any number of clients on the same local network.Multi-Channel Rooms — Dynamic network partitioning via /join <room>. Messages are safely scoped to your active channel.Direct File Transfer — Peer-to-peer data plane offloading. Files transfer in 4KB chunks through temporary ephemeral sockets, bypassing and protecting the core server.Connection Resilience — Automatic background fault detection with an exponential backoff retry loop (1s, 2s, 4s, 8s, 16s) to keep clients alive through server drops.Private DMs — Secure private messages via /msg <username> <text>.Live UI Workspace — Online users sidebar tracking client state, tab autocomplete for usernames, and color-coded message styling.Daily Log Rotation — Automated, thread-safe session auditing saved cleanly to logs/chat_YYYY-MM-DD.log.Server Admin Commands — Complete console oversight using /kick, /list, and /broadcast.How to runPrerequisitesJava 17 or higherGit Bash (Windows) or any terminal (Mac/Linux)Start the serverBash./run.sh server
